Spreadsheet API

This Spreadsheet API project gives you a URL that acts as a web API for a spreadsheet. You can fetch data, append and update rows, and set fine-grained security options.

The API endpoint for this project is: https://api.cloudstitch.com/papodaca/spreadsheet-powered-api/datasources/sheet
